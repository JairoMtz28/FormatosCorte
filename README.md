# FormatosCorte

Este proyecto es una aplicación desarrollada en **Visual Basic .NET** utilizando **Windows Forms**, creada con el objetivo de **digitalizar y optimizar el control del formato "Hora por Hora"** utilizado en las máquinas de corte de la empresa.

## Descripción

Anteriormente, el registro del **Hora por Hora** se realizaba de forma manual en papel, lo que implicaba un proceso más lento, propenso a errores y con dificultad para acceder a la información histórica.  
Con este programa, **todo el proceso se lleva a cabo de manera digital**, permitiendo registrar, consultar y almacenar los datos directamente en una **base de datos MySQL**.

## Características principales

- **Digitalización completa del formato Hora por Hora.**  
- **Conexión directa con una base de datos MySQL.**  
- **Interfaz amigable y fácil de usar**, desarrollada en Windows Forms.
- **Búsqueda y consulta de registros anteriores** de manera rápida y sencilla.
- **Mayor control, seguridad y respaldo de la información.**

## Tecnologías utilizadas

- **Visual Basic .NET (Windows Forms)**
- **MySQL**
- **.NET Framework**

## Objetivo

Modernizar el control operativo de las máquinas de corte mediante la **digitalización de los formatos físicos**, reduciendo el uso de papel, mejorando la trazabilidad de los datos y optimizando la gestión del tiempo de producción.
